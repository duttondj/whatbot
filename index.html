Hoes

What is it?

Bitches is an IRC bot written for #indie. It is written in Ruby and relies on Cinch. Hoes is an older fork of bitches by britishtea.

Installation

  1. Clone the repository: git clone git://github.com/moftz/hoes.git && cd hoes.
  2. Read INSTALL to learn how to setup hoes.
  3. Run it: ruby -I lib app.rb.

Note: You'll need Ruby 1.9.3 or higher and git.
